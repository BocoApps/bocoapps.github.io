Description

My Database is a versatile application for smartphones designed to manage customizable SQLite databases with integrated image support. It allows users to create, edit, and organize multiple databases, tables, and records (posts) containing text fields and attached images.

Key features include database encryption with passwords, importing/exporting data in ZIP (database + images), pure SQLite DB, or CSV formats, keyword search with advanced filters (contains, equals, greater/less than, between) and sorting, image handling (uploading from gallery/camera with EXIF rotation, compression, thumbnails, full-screen viewing with zoom and swipe navigation), and customizable settings like light/dark themes, image quality (high/low compression), thumbnail scaling (50%â€“300%), and option to show all posts on table open.

The app supports pagination for efficient handling of large datasets, isolated background processing for imports/exports with progress indicators, column management (add, rename, delete, reorder), and exporting search results as CSV. Ideal for personal data management, inventory tracking, or visual note-taking, the app ensures secure, offline storage with robust error handling and permissions management.
